# COMP 412: Compilers
Private repository for COMP 412 Fall 2017 at Rice University.

## Lab 1
**Due 09/15/2017**

You	will build a local register	allocator--that	is,	a	program	that takes as input	a	branch-free
sequence of operations and	produces as	output an	equivalent sequence	of operations that uses a	
specified	number of registers.
